If not already installed
========================
npm install -g bower gulp yo generator-ng-poly
npm install -g tsd

In frontend folder
==================
npm cache clean
npm install
bower install


Run the frontend:
1. Start the server
2. Load the frontend project in webstorm
3. Run gulp build
4. Load http://localhost:3000/#/home (Configure a webstorm debug configuration with this url for index.html). 

Prerequistie to run the e2e test
================================

1. Install silenium standalone:
gulp webdriverUpdate

2. Download and install java.
